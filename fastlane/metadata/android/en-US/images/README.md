# Image Placeholders

This directory contains placeholder files for F-Droid and Play Store images.

## Required Images

- **icon.png** (512x512): App icon
- **featureGraphic.png** (1024x500): Feature graphic for store listing
- **phoneScreenshots/**: Directory for app screenshots (max 8 images)

## Instructions

1. Add your app icon as `icon.png` (512x512 pixels)
2. Create a feature graphic as `featureGraphic.png` (1024x500 pixels)
3. Add screenshots to the `phoneScreenshots/` directory
4. Screenshots should be high-quality PNG images showing key app features

These images will be used by F-Droid and can also be synced to Google Play Store using fastlane.
